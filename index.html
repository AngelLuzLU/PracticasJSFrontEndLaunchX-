Para buscar un pokemon, ingrese su número de pokédex o el nombre del pokémon.

Puede cambiar de vista haciendo uso de los botones de la parte inferior izquierda.

Si puede ver a psyduck confundido, quiere decir que ha ocurrido algún error al buscar la información del pokémon, verifique haber escrito el nombre correctamente; si el error persiste, puede que el servicio no se encuentre disponible.

Los movimientos con nivel 0 se aprenden por medios especiales como eclosión, TM/HM, tutores, etc.

Gracias por usar esta pokédex!
